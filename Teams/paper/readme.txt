-namestena skripta za kompresiju
-namestena skripta za generisanje labela

# H(Hue) #
-represents the color type

# S(Saturation) #
-represents the vividness or purity of the color
-higher values indicate more vivid colors
-lower values indicate more grayscale or less saturated colors
-increase the lower threshold to detect more vivid colors
-narrow down the saturation range to focus on less vivid colors


# V(Value) #
-represents the brightness or intensity of the color. 
-higher values indicate brighter colors
-lower values indicate darker colors
-increase the lower threshold to detect brighter colors
-narrow down the value range to focus on darker colors
